项目介绍：

CodePrism是一个面向代码规范检查的智能助手系统。它利用大语言模型的推理能力，帮助用户检查C代码是否符合编码规范，识别违规代码，并提供修改建议。
本项目为基础框架版本，包含了AI调用、Web界面、数据存储等核心功能，学生可以在此基础上进行扩展和优化。

==============================================================================

运行指导

python backend.py
访问 http://localhost:5000 开始使用

==============================================================================

个性化设计

打开 static/css/style.css，找到"配色方案切换区"，注释当前方案，取消注释想要的方案即可。
提供的配色方案：
科技蓝（默认）- 专业、现代
优雅紫 - 高雅、独特
森林绿 - 自然、舒适
暖橙色 - 活力、温暖
极简灰 - 简约、专注
自定义 - 自由定制

扩展指南

1. 优化Prompt

在 ai_module.py 中找到 system_prompt，添加你的提示词：
self.system_prompt = "你是一个很厉害的代码规范分析大师。"
# 添加更多指导
self.system_prompt += "\n\n在分析代码时，请按照以下步骤进行："
self.system_prompt += "\n1. 仔细阅读代码，理解其功能"
self.system_prompt += "\n2. 对照每一条规范逐一检查"
# ...更多提示词
