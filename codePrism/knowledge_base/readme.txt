Example-Suite.rarMISRA代码示例集

简介
====
本数据集来源于MISRA C:2012官方示例，用于训练和验证代码分析AI系统。该示例集展示了常见的C语言编码规范问题和最佳实践。
在本项目中，用作知识库。

数据集特点
=========
1. 权威性：来自MISRA官方发布的示例
2. 结构化：每个规则对应独立文件
3. 双重示例：包含合规和不合规的代码示例
4. 完整性：覆盖MISRA C:2012主要规则

文件组织
=======
- R_xx_yy.c: 规则示例文件（如R_20_04.c表示规则20.4）
- D_xx_yy.c: 指令示例文件

适用场景
=======
- 代码质量分析
- 安全性检查
- 合规性验证
